Database Project – Infrastructure Management
Жобаның мақсаты
Бұл деректер қоры серверлік инфрақұрылымды басқару жүйесін сипаттайды.
Негізгі кестелер:
datacenters – дата орталықтары
servers – серверлер
admins – әкімшілер
services – қызметтер
incidents – инциденттер
Байланыстар:
Бір датаорталықта бірнеше сервер болады (1:N)
Бір серверде бірнеше инцидент болуы мүмкін (1:N)
ДҚ қасиеттері:
3NF нормализация
PRIMARY KEY, FOREIGN KEY
CHECK, UNIQUE, NOT NULL
